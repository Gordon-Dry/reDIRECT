reDIRECT v0.9.4b. 
PHASE TWO

KSP version: 1.4.2 (1.3.1+ supported)

Release Date: 09-05-2018.

Created by BENJEE10. 

Thank you for downloading reDIRECT, a revamp of the DIRECT mod which introduces shuttle-derived parts for Kerbal Space Program. 

This release includes parts required for a Shuttle stack and the Jupiter family of rockets, as well as the Orion spacecraft.  

INSTALLATION:

Simply drag and drop the "reDIRECT" folder into your KSP gameData directory. 

DEPENDECIES:

B9PartSwitch and Community Resource Pack are bundled with this mod. You may need to update B9PartSwitch for the mesh switching features to function in newer versions of Kerbal Space Program.
